STRINGS

Strings in python are a sequence of characters surrounded by either single quotation marks, or double quotation marks.


"python" is the same as 'python'



Strings can be displayed with the print() function


Multi line strings are assigned with three quotes like in the example below

Example of multiline strings


a = """Lorem ipsum dolor sit amet,

consectetur adipiscing elit,

sed do eiusmod tempor incididunt

ut labore et dolore magna aliqua."""

print(a)



The length of a string can be obtained using the len() function


Let us try that in the console below.

We can also combine two or more strings. This is known as concatenation.


Strings are arrays


Strings store each character(including white space) in an array which represent the byte in unicode. Therefore, each character in the string can be accesed individually using its index(in square brackets).


NB: array indexes start from zero.


Here is an example

Example of multiline strings


x = "QWERTY"

print(x[2])



The above example will print out "E". Remember, index starts from zero so "Q" wil have an index of zero,"W" will be at index 1 and then "E" will be at index 2 and so on.
