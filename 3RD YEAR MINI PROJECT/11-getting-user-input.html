GETTING USER INPUT

In previous chapters, we have leanrt how to display text and other results to the user using the Print() function. In this chapter we are going to look at how to take user input and use it for various computations and other purposes.

Taking user input in python is simple. We use the input() function to achieve this purpose

The input() function is a built-in function in python just like the print() function. This means that the user does not have to create the function himself but will just have to call it when needed.

The parameter we put inside the input function is the text we want to display to the user Once the user presses the enter key, the value is returned. We typically store user input in our variable so that we can use the information in our program.

Lets demostrate with an example

Example of user input code


first_name = input("What is your first name")

sur_name = input("What is your surname")

#Lets concatenate to get the full name

full_name = (first_name + " " + sur_name)

print(full_name)


Lets try the above example in the editor and see the output


NB:Please take note that python returns the user input as a string even if the user enters a number and therefore appropriate type casting has to be done when necessary. When yhe user inputs a number and it is not converted to the appropriate number type, it could lead to complications when we attempt to do calculations with them.


Lets look at another example to demonstrate that.

Example of user input code


dob = int(input("What is your year of birth"))

current_year = 2023

#Lets calculate your age

user_age = (current_year - dob)

print("You are" + " " + user_age + " years old")



We realise that when we run the code an error is generated which say a string and integer type cannot be concatenated. We fix this Error by using the str() function to convert the user_age variable to a string.


So now,go back to the code,make that change and viola!!!! you are good to go.
