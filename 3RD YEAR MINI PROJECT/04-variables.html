VARIABLES

Think of a variable as a container that stores something. for example, a cup for storing your coffee, or a bowl for holding the rice your about to eat,a bucket for storing water which you'll use to bath later, or even your fridge which stores a lot of items.


Python variables are simply memory locations that hold information, such as numbers or text, that the python code can later use to perform tasks .


Python has quite a number of variables which are listed below but in this chapter we will only consider the first four(4).


  • integer
  • float
  • string
  • boolean
  • list
  • dictionary
  • tuple
  • set

Before we begin learning about variables, let's take note of the this. Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.


1. Integer

A python integer or int refers to whole numbers that can be either positive or negative.


Example of int variable


a = 1

b = -200

c= 3500

To print the above code , we add these three lines to the ones above

print("a =", a)

print("b =", b)

print("c =", c)

2. Float

A python float variable allows you to store floating-point values or decimals.


Example of float variable


x = 5.7

y = 89.99

z= 3.67

To print the above code , we add these three lines to the ones above

print("x =", y)

print("y =", y)

print("z =", z)



3. String

A string variable in Python allows us to store letters, words, or sentences. To declare a string variable, you must write the letter, word, or sentence in double quotes. The double quote is very important. without it, your string declaration is invalid and may generate a syntax error.


Example of string variable


name = "Johnny"

full_name = "Johnny English"

position= "CIA detective"

To print the above code , we add these three lines to the ones above

print("name =", name)

print("full_name =", full_name)

print("Position =", Position)



Taking a careful look at the above code, you'll see that the variable position was capitalized in the print() function but this will not affect the code in any way since we iniatially established the fact that python is not case-sensitive.



4. Boolean

A boolean variable stores only two values, True or False. We use them for logical operations, like checking if a condition is true or false. We don't enter the true or false in quotes for declaring a boolean as we do for strings.


Example of boolean variable


x = true

y = false

To print the above code , we add these three lines to the ones above

print("x =", y)

print("y =", y)

print("z =", z)
