DATA TYPES

In a previous chapter, we learnt about variables and it was said that they are containers for storing data


Variables store different types of data and these are what we refer to as data types



They are into categories and they are summarized in the table below


text type string
numeric types int float complex
sequence type list tuple range
mapping type dict
boolean type bool

Getting The Data Type


You can get the data type of any variable by using the type() function


Example


Print the data type of the value y

y = 8

print(type(y))
