PYTHON SYNTAX

Syntax refers to the rules that define the combinations of symbols that are considered to be correctly structured statements or expressions in that language


Python has a very simple syntax which makes it an ideal language for beginners in programming


  • It is not case-sensitive. This means that "print", "Print" and "PRINT" are all considered to be the same even though capitalization has made them look different. Python does not respect capitalization.

  • It doesn't use a semi-colon at the end of each statement. Languages like java and c++ use semi-colons to mark the end of an executable statement. Omitting the semi-colon in such languages leads to generation of syntax error but python does not do that.

    • PYTHON INDENTATION

    Indentation refers to the spaces at the beginning of a code line.


    Where in other programming languages the indentation in code is for readability only, the indentation in Python is very important.


    Python uses indentation to indicate a block of code.


    Example


    if 5 > 2:

        print("Five is greater than two!")



    Python will give you an error if you skip the indentation


    Example


    syntax error


    if 5 > 2:

    print("Five is greater than two!")



    The number of spaces is up to you as a programmer, the most common use is four, but it has to be at least one.
    Pressing the tab key to indent is also helpful as this gives you a reasonable amount of white space


    Example


    if 5 > 2:

        print("Five is greater than two!")

    if 5 > 2:

            print("Five is greater than two!")



    You have to use the same number of spaces in the same block of code, otherwise Python will give you an error:


    Example


    syntax error


    if 5 > 2:

       print("Five is greater than two!")

            print("Five is greater than two!")
